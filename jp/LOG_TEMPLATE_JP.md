# 📝デイリーログ【テンプレート】

フォーマット：
- YYYY-MM-DD
- Level: 1 / 2 / 3
- プロジェクト / リポジトリ
- 内容: 何をしたか簡単に記述
- PR / Issue / リンク: 該当する場合はURL
- メモ / 振り返り: 任意（1〜2行）

例：
```
### 2026-03-30
- **Level:** 1  
- **プロジェクト / リポジトリ:** [Python](URL)  
- **内容:** READMEの誤字を修正し、使用例をわかりやすく改善  
- **PR / Issue / リンク:** [PR #12](URL)  
- **メモ / 振り返り:** 小さな修正だが、初心者の理解に役立つ  

### 2026-03-31
- **Level:** 2  
- **プロジェクト / リポジトリ:** [HuggingFace](URL)  
- **内容:** エンティティ認識関数のテストケースを追加  
- **PR / Issue / リンク:** [PR #45](URL)  
- **メモ / 振り返り:** BIOタグ変換について理解が深まった  
```
