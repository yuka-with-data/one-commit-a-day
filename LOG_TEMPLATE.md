# Contribution Log — One Commitment a Day

This is my daily log for the One Commitment a Day project.  
Each entry should be short, record what I did, and include links if applicable.  

**Rules:**  
- Minimum: Level 1 contribution per day  
- Optional: Level 2–3 contributions  
- Optional: Private repo work noted separately at the end  

---

## 📅 Daily Log Template

### YYYY-MM-DD
- **Level:** 1 / 2 / 3  
- **Project / Repo:** [Repo Name](URL)  
- **Action:** Short description of contribution  
- **PR / Issue / Link:** URL if applicable  
- **Notes / Reflection:** Optional, 1–2 lines  

**Example:**
```
### 2026-03-30
- **Level:** 1  
- **Project / Repo:** [Python](URL)  
- **Action:** Fixed typo in README and clarified example usage  
- **PR / Issue / Link:** [PR #12](URL)  
- **Notes / Reflection:** Small fix, but helps newcomers understand setup  

### 2026-03-31
- **Level:** 2  
- **Project / Repo:** [HuggingFace](URL)  
- **Action:** Added a test case for entity recognition function  
- **PR / Issue / Link:** [PR #45](URL)  
- **Notes / Reflection:** Learned more about BIO tagging conversion  
```

### ⚡ Optional: Private Repository Work
- Only used if no public contribution possible that day  
- Example format:

### 2026-04-01
- **Level:** 1 (Private)  
- **Project / Repo:** Personal AI Agent Project  
- **Action:** Added example input validation  
- **Notes / Reflection:** No public repo today, but still made progress
